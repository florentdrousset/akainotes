#### Dired

<!--list-separator-->

-  Cheatsheet

    -   SPC o - : jump to current dir
    -   Ctrl-x D : open Dired
    -   M : mark file
    -   T : toggle mark / unmarked files
    -   D : delete marked files
    -   R : rename marked files
    -   Z : compress
    -   U : undo marking
    -   Shift-U : undo all markings
    -   Meta-Enter : preview file
